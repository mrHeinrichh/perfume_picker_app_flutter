# Perfume Picker App User Guide

## Browsing

When the app opens, everyone starts as a normal user. No registration is required.

Use the homepage filters to choose:

- Gender: Male, Female, or Unisex
- Fragrance characteristics: woody, fruity, citrus, floral, fresh, spicy, amber, musky, aquatic, and related scent styles
- Notes: top notes, middle notes, and base notes

After selecting filters, open the results page. Perfumes with the most matching filters appear first.

## Perfume Details

Tap a perfume from the results list to view:

- Name
- Description
- Product image
- Gender category
- Fragrance characteristics
- Top notes
- Middle notes
- Base notes

## Admin Mode

Admin mode is only for managing products.

Admin credentials:

```text
Username: admin
Password: Admin@1234
```

You can also use biometrics or Face ID as a shortcut if the device supports it and has biometrics enrolled.

## Product Management

After logging in as admin, you can:

- Add perfume products
- Edit existing products
- Delete products
- Select product images from the device gallery

Products are stored in memory for this simple app version, so changes are local to the running app session.
